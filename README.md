# Node.js Project

A Node.js project initialized with pnpm.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (v7 or higher)

### Installation

1. Clone the repository
2. Install dependencies:

```bash
pnpm install
```

### Running the Project

To run the project:

```bash
node src/index.js
```

## Project Structure

```
.
├── src/           # Source files
│   └── index.js   # Main entry point

```
